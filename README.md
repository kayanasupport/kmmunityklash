# K'mmunity Klash (Family-Feud Parody)
Single-tab game board for Google Meet with host controls, two teams, buzzer/strikes, round multipliers, and Google-Sheet–driven question packs.

## Quickstart
- `npm i`
- Add `VITE_SHEET_ID` to `.env` (published CSV).
- `npm run dev`

## Host Shortcuts
1–8 reveal/hide answers • A/B buzz team • R reset buzz • S/D +strike A/B • W/E award to A/B

## Theme
Change the title and font in the right-side Controls; font defaults to **Bangers** (cartoony show look). Replace Google Fonts or add your own by editing `index.html`.
